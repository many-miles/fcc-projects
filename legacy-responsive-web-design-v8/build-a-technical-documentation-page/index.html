Variables and Data Types

Variables store data values, and data types determine the kind of values.

Common data types include numbers, strings, booleans, objects, and arrays.

let x = 10; const name = "Alice";
  • Number
  • String
Functions

Functions are blocks of code designed to perform specific tasks.

They can accept parameters and return values.

function sum(a, b) { return a + b; } const greet = () => "Hello";
  • Declaration
  • Expression
Loops

Loops allow repeating a block of code multiple times.

Common loops include for, while, and do-while loops.

for(let i=0; i<5; i++){console.log(i);}
  • for loop
  • while loop
Events

Events allow code to respond to user actions like clicks or keyboard input.

Event listeners are used to handle these events.

button.addEventListener('click', handleClick);
  • click
  • mouseover
